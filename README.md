# FuzzyPaste

fuzzy search でコピー履歴を瞬時に呼び出す、Mac アプリ。

## ダウンロード

[![Download app for macOS](Resources/download-macos.svg)](https://github.com/gehnmaisoda/fuzzy-paste/releases/latest/download/FuzzyPaste.dmg)

> 初回起動時は .app を右クリック →「開く」で起動してください。

## ハイライト

- **Fuzzy Search** — 数文字タイプするだけで履歴を瞬時に絞り込み
- **テキスト・画像・ファイル対応** — テキストはもちろん、画像や PDF・CSV・ZIP などあらゆるファイルの履歴を管理
- **OCR 検索** — 画像内のテキストを自動認識し、fuzzy search の対象に
- **スニペット** — 定型文をタグ付きで整理。動的プレースホルダーやドロップダウン選択肢にも対応
- **Frecency ランキング** — よく使うアイテムが自動的に上位に表示
- **マルチセレクト** — 複数アイテムを選んで一括ペースト
- **Quick Look** — PDF ビューアー・CSV テーブルビューなど、コンテンツに応じたリッチプレビュー
- **ドラッグ & ドロップ** — 検索結果から直接ほかのアプリへ D&D
- **CLI ツール (`fpaste`)** — ターミナルから履歴検索・スニペット管理

## 動作環境

- macOS 13 (Ventura) 以降

## 使い方

### 基本操作

| 操作 | 説明 |
|---|---|
| `Cmd+Shift+V` | 検索ウィンドウを表示（ホットキーはカスタマイズ可） |
| 文字入力 | fuzzy search でリアルタイム絞り込み |
| `↑` `↓` | 結果を選択 |
| `Enter` | 選択アイテムをペースト |
| `Cmd+C` | クリップボードにコピー（ペーストせず） |
| `Shift+Space` | Quick Look プレビュー |
| `Shift+Click` / `Cmd+Click` | マルチセレクト |
| `Esc` | ウィンドウを閉じる |

### タグフィルタリング

検索ウィンドウで `Tab` キーを押すとタグフィルタモードに切り替わり、タグで絞り込めます。複数タグの AND 検索にも対応。画像・PDF・CSV などには自動タグが付くので、`Tab` → `img` で画像だけ一覧、といった使い方ができます。

### スニペット

よく使う定型文をスニペットとして保存できます。タグを付けて整理し、検索結果に履歴と一緒に表示されます。

**動的スニペット** — プレースホルダーを使ったテンプレートを作成可能。ペースト時に入力ダイアログが表示され、リアルタイムプレビューで確認しながら値を埋められます。

- `{{名前}}` — 自由入力
- `{{OS:macOS,Windows,Linux}}` — ドロップダウンから選択

**自動タグ** — 画像・PDF・CSV・JSON などのコンテンツには種別タグが自動で付与されます。

**履歴からスニペット登録** — クリップボード履歴のアイテムをそのままスニペットとして保存できます。

スニペットは Markdown + YAML frontmatter 形式で `~/.config/fuzzy-paste/snippets/` に保存されるため、お好みのエディタで直接編集することもできます。変更は自動で検知・反映されます。

### CLI ツール

`fpaste` コマンドでターミナルからも操作できます。

```sh
fpaste list                  # スニペット一覧
fpaste search "検索語"        # fuzzy search
fpaste add "テキスト"         # スニペット追加
fpaste history list          # 履歴一覧
fpaste history add "テキスト" # 履歴に追加
fpaste export snippets.zip   # スニペットをエクスポート
fpaste import snippets.zip   # スニペットをインポート
```

## 設定

メニューバーアイコンから設定ウィンドウを開けます。

- **起動ホットキー** — 任意のキーコンビネーションに変更
- **ウィンドウサイズ** — 小・中・大の 3 段階
- **履歴保持件数** — 100〜2,000 件
- **除外アプリ** — パスワードマネージャーなど、特定アプリからのコピーを履歴に保存しない
