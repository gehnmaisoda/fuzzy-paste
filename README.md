# FuzzyPaste

> macOS ネイティブのクリップボードマネージャー。**爆速の fuzzy search** でコピー履歴を瞬時に呼び出す。

[Clipy](https://github.com/Clipy/Clipy) にインスパイアされつつ、モダンな UI と豊富な機能を備えた次世代クリップボードツール。

## ダウンロード

[![Download app for macOS](Resources/download-macos.svg)](https://github.com/gehnmaisoda/fuzzy-paste/releases/latest/download/FuzzyPaste.dmg)

> 署名なしのため、初回起動時は .app を右クリック →「開く」で起動してください。

## ハイライト

- **Fuzzy Search** — 数文字タイプするだけで履歴を瞬時に絞り込み
- **テキスト・画像・ファイル対応** — テキストはもちろん、画像や PDF・CSV・ZIP などあらゆるファイルの履歴を管理
- **スニペット** — 定型文をタグ付きで整理。`{{名前}}` のような動的プレースホルダーでテンプレートとしても使える
- **マルチセレクト** — 複数アイテムを選んで一括ペースト
- **Quick Look** — `Shift+Space` でアイテムをその場でプレビュー
- **ドラッグ & ドロップ** — 検索結果から直接ほかのアプリへ D&D
- **CLI ツール (`fpaste`)** — ターミナルから履歴検索・スニペット管理

## 動作環境

- macOS 13 (Ventura) 以降

## 使い方

### 基本操作

| 操作 | 説明 |
|---|---|
| `Cmd+Shift+V` | 検索ウィンドウを表示（ホットキーはカスタマイズ可） |
| 文字入力 | fuzzy search でリアルタイム絞り込み |
| `Enter` | 選択アイテムをペースト |
| `Cmd+C` | クリップボードにコピー（ペーストせず） |
| `Shift+Space` | Quick Look プレビュー |
| `Esc` | ウィンドウを閉じる |

### スニペット

よく使う定型文をスニペットとして保存できます。タグを付けて整理し、検索結果に履歴と一緒に表示されます。

**動的スニペット** — `{{日付}}` `{{宛名}}` のようなプレースホルダーを含むテンプレートを作成可能。ペースト時に入力ダイアログが表示され、リアルタイムプレビューで確認しながら値を埋められます。

スニペットは JSON 形式で **インポート/エクスポート** にも対応。チームでの共有や別マシンへの移行も簡単です。

### CLI ツール

`fpaste` コマンドでターミナルからも操作できます。

```sh
fpaste list                  # 履歴一覧
fpaste search "検索語"        # fuzzy search
fpaste add "テキスト"         # 履歴に追加
fpaste export snippets.json  # スニペットをエクスポート
fpaste import snippets.json  # スニペットをインポート
```

## 設定

設定ウィンドウから以下をカスタマイズできます。

- **起動ホットキー** — 任意のキーコンビネーションに変更
- **ウィンドウサイズ** — 小・中・大の 3 段階
- **履歴保持件数** — 100〜2,000 件
- **除外アプリ** — パスワードマネージャーなど、特定アプリからのコピーを履歴に保存しない
